# Pipeline de Dados de Vendas – ETL

## Descrição
Projeto de Engenharia de Dados com foco em ETL utilizando Python e SQL Server.

## Tecnologias
- Python
- Pandas
- SQL Server
- Git

## Fluxo do Projeto
1. Extração de dados via CSV
2. Transformação (limpeza, padronização e cálculo)
3. Carga em banco de dados relacional

## Objetivo
Demonstrar conceitos fundamentais de Engenharia de Dados:
- ETL
- Modelagem
- Pipeline de dados
- Organização de código

1. Criar o banco executando `create_tables.sql`
